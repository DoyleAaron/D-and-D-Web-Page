## 🔧 LINKING RULES

1. **Full names first** — Match "Bridge-Warden Elara Thornguard" before "Elara"
2. **No self-links** — Don't link a character's name in their own file
3. **Skip already-linked** — Don't create `[[already linked](path)](path)` 
4. **Case-insensitive matching** — "kadena" should still link to Kadena
5. **Word boundaries** — "Korrin" shouldn't match inside "Korgin"
6. **Skip headers and tables** — Don't link text inside `#`, `|`, or `**bold**` markers
7. **URL encode spaces** — `Mortling Stronghold.md` → `Mortling%20Stronghold.md`
