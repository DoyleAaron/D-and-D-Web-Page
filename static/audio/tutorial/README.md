## Tips for Recording

1. **Voice:** Friendly, slightly witty female elf - welcoming but with a bit of playful sass
2. **Tone:** Like a bartender who knows all the regulars
3. **Pace:** Moderate - not too fast, let the words breathe
4. **Character greetings:** Each character should feel like she knows them personally
5. **Export:** MP3, 128kbps is fine for voice
